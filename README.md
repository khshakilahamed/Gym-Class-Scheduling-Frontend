### Installation Process

- Install all dependencies

```bash
yarn add
# or,
npm install
```
- .env configure
```
NEXT_PUBLIC_API_BASE_URL=https://gym-schedule-backend-eight.vercel.app/api/v1
```

- Run Project

```bash
yarn dev
# or,
npm run dev
```
